# redis可视化管理工具

软件特点:
1. 多样化管理服务器,支持服务器的密码认证服务.
2. 支持所有redis的数据类型,例如: string,hash,set,zset,list.
3. 支持单个key的重命名功能.
4. 支持单个key的删除功能.
5. 支持查看key数据的大小,引用等辅助属性.
6. 列表类的数据可分页,加快查看速度,不卡死.
7. 支持生命周期的修改以及移除功能
8. 对key包增加了支持, 查看更直观,更清爽.
